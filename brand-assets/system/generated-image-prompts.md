# Generated Image Prompts

## Base Prompt

Use this as the baseline prompt for backgrounds, thumbnails, and branded generated images:

```text
abstract editorial-tech composition, geometric panels, grid field, directional lines, restrained data texture, paper and ink neutrals, electric blue accent, premium teaching brand, clear negative space, no people, no robots, no circuits, no glowing brain, no generic stock AI imagery
```

## Variants

### Hero Background

```text
abstract editorial-tech hero background, large layered panels, grid field, directional lines, premium technical composition, dark ink background, sparse electric blue accents, generous negative space for headline text, no people, no robots, no circuits
```

### Section Divider

```text
abstract editorial-tech section divider, clean geometric fields, calm structure, lighter paper background, sparse signal-blue highlights, directional layout energy, minimal texture, no stock AI motifs
```

### Thumbnail Background

```text
high-contrast editorial-tech thumbnail background, geometric depth, dark ink base, bright electric blue accent shapes, structured energy, strong negative space for large type, no humanoids, no cyberpunk skyline, no glowing brain
```

## Banned Motifs

Never use prompts that encourage:

- robots
- brains
- chips or motherboard wallpaper
- hologram hands
- futuristic cityscapes
- purple neon haze
- cinematic humanoid AI scenes

## Normalization Checklist

After generating an image:

1. choose only the clearest and calmest option
2. crop to the approved composition
3. reduce excessive saturation
4. shift colors back toward `Ink + Signal`
5. add a subtle overlay if the image competes with text
6. export only after it matches the rest of the system
