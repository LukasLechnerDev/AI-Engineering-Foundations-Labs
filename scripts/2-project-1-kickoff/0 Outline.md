# Outline

## Teaching goal
- Kick off the portfolio project and set the story for the rest of the course.

## What to cover
- what the project is and why it is a good first AI engineering portfolio project
- what the final user value is
- what the pipeline does end to end
- how we will build first and add theory when we need it
- where later modules fit into the project
  - prompt engineering when output quality becomes the bottleneck
  - evaluations when we need evidence that changes helped
  - observability when we want to inspect traces and failures
  - productionizing when we need to make the system robust
  - deployment when the product is ready to ship

## Note
- This is where module transitions should live.
- "Bridge to later modules" should not be a standalone lesson.
