## What is this course covering?

In this course, we will together build and deploy your first simple AI-powered app.
It's a job posting analyzer that extracts useful information like the required skills from a job posting.
The app is kept simple on purpose, but it has all the properties of a real production app.
It talks to an LLM, is deployed, has evaluations, and we also have set up monitoring and a CI/CD pipeline.

This course should helps you to become an AI Engineer in the fastest way possible.
And the fastest way to get an AI engineering job is to work on your own AI project.

This course focuses less on the "fancy stuff" - like image generation etc. 
but on the things that are important in production apps.

By deploying your own portfolio apps.
We will build a real, production grade AI application together.

It's not a demo or toy app, it also contains
- evals that check if the app behaves as expected
- rate-limiting and monitoring
- monitoring

You will learn what makes a good portfolio project and I will give you
plenty of project ideas and also show you how to market them well, for instance by documenting
implementation details in the Readme of the github repository.
