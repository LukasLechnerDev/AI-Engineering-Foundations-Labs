# Outline

## Teaching goal
- This module is about packaging and shipping the product once it already works locally.

## What to cover
- Dockerizing the project
- choosing an AWS service
- environment variables and secrets
- scheduled jobs and containers
- light CI/CD

## Keep out of this module
- model routing
- retries and fallbacks
- parsing and validation strategy
- other runtime reliability concerns that belong in Productionizing AI Systems
