### Introduction

Hi, and welcome to this course! I am very happy to see you here!

This is a practical course, and therefore, we want to not start with theory but start to interact with LLMs as quickly as possible. 

But before we can do that we need to first set up our development environment. 

As I said in the preview video, the goal of this course is to transform you into an AI Engineer, and the best way to achieve this is for you to get hands-on and start implementing your own AI-based projects and products. You really stand out if you can show your production products to recruiters and companies that want to hire you. 

Therefore, I highly recommend that you don’t just passively consume this course, but actively work together with me, play around with the code and build your own projects on the side. 

Throughout this course, you will learn new concepts with either small code examples that are contained in Jupyter Notebooks, or by using real Python code to build real Python apps. 

I am personally using Visual Studio Code, and I guess a lot of you guys will also use it, but of course you could use another VS Code based editor like cursor or any other IDE like pycharm. 

### Cloning the Repo

The first task that we need to do is to download the repository of this course, which contains all the labs, so the code examples of this course. 

> Show Repo on Github

So let's copy the URL of that repo into our clipboard. 
We can use the terminal to clone the repo, of course, but another convenient way is to simply click on “Clone Git Repository” here in VS Code, then we just have to log in to Github, paste in the repo url, select the folder in which the repo should be saved, and afterwards we can direcly open up that project in VS Code.
