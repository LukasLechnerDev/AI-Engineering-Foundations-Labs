# 90 - Raw SDK vs Frameworks

- Explain why the course used the raw SDK first and set up the later LangChain refactor from a position of understanding.

## What to cover
- why the course used the raw SDK first
- what students understand now because they built the raw version themselves

- when a small project should stay simple
- which repetition or complexity would justify a framework refactor
- why LangChain is the next concrete comparison before RAG
- when a bigger codebase may benefit from abstractions like LangChain or PydanticAI
